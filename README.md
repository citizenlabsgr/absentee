# Requesting an Absentee Ballot in Michigan

1. Check to see if you already have absentee voting status: https://mvic.sos.state.mi.us/
2. Download the fillable form: https://www.michigan.gov/sos/0,4670,7-127-1633_8716_8728-21037--,00.htm 
3. Locate your Clerk's email address: https://mvic.sos.state.mi.us/Clerk
4. Email them the completed form
5. Wait a few days
6. Verify you now have absentee voting status: https://mvic.sos.state.mi.us/
